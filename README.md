# Arena — AI Model Battle Platform

A community-powered AI model evaluation and benchmarking platform.

## Setup

1. Add your API keys to `.env`:
   - `GEMINI_API_KEY` (Handled by AI Studio)
   - `OPENAI_API_KEY`
   - `ANTHROPIC_API_KEY`
   - `GROQ_API_KEY`

2. The app uses SQLite for the database.
3. Run `npm run dev` to start the development server.
